Bug Tracker

Bug Tracker is a software defect tracking system. Commonly known as an issue tracker. Bug Tracker is built with ASP.NET, MVC, C# and PostgresSQL. A role based security system was implemented to create permission based roles. The roles of Admin , Project Manager, Developer and Submitter restrict what the user can see and do. The software allows for users to submit tickets and those tickets are tracked through status from Open to Complete.

The system was built with C# and .Net in 3 weeks. The front end was developed with HTML, Javascript and Bootstrap. The back-end utlizies MVC and C#. The database was built with PostgresSQL Server and hosted on Heroku. Please visit the site and try it out.

  • Project Type:

    Web Application

  • Technologies:

    C#, PostgresSQL, ASP.NET MVC, JS, HTML5/CSS, Bootstrap, Heroku

Blog

This is my personal blog site built with ASP.NET, MVC, C# and PostgresSQL. A role based security system was implemented to create permission based roles. The roles of Admin, and Moderator restrict what the user can see and do. The software allows for anonymous users to read about blog posts by the Admin; also, they are allowed to navigate the pages 'About Me' and 'Contact Me'. The user can register to the website in order to put comments about the blogpost. The moderator can moderate comments and also delete it if necessary. Admin role is for the site owner which can do everything such as: create, edit, delete posts and comments.

The system was built with C# and .Net in 3 weeks. The front end was developed with HTML, Javascript and Bootstrap. The back-end utlizies MVC and C#. The database was built with PostgresSQL Server and hosted on Heroku. Please visit the site and try it out.

  • Project Type:

    Web Application

  • Technologies:

    C#, PostgresSQL, ASP.NET MVC, JS, HTML5/CSS, Bootstrap, Heroku

Address Book

This is address book app built with ASP.NET, MVC, C# and SQL. It is utilizing Microsoft's ASP.Net Identity for authentication. This is where I started honing my skills in securing a web-application through providing users a secure way to log in the system. Once the user is logged in, the user can add and manage contacts as well as categories for it. The user can also send an email to a contact and also to the whole group of people under one category.

The system was built with C# and .Net in 1 week. The front end was developed with HTML, Javascript and Bootstrap. The back-end utlizies MVC and C#. The database was built with PostgresSQL Server and hosted on Heroku. Please visit the site and try it out.
